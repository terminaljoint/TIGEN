Quick Start

Get TIGEN running in seconds.

1. Clone the Repository

$ git clone https://github.com/terminaljoint/TIGEN.git
$ cd TIGEN

2. Start a Local Server

$ python3 -m http.server 8000

3. Open in Browser

http://localhost:8000

4. Basic Usage

import { Engine } from './main.js';

const engine = new Engine();
engine.run();
